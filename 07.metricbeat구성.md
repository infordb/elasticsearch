
## 모니터링 대상 nginx 설치 및 설정

https://holjjack.tistory.com/114


etc/yum.repos.d/ 경로에 nginx.repo 파일을 추가한다. 
```
[nginx] 
name=nginx repo 
baseurl=http://nginx.org/packages/centos/7/$basearch/ 
gpgcheck=0 
enabled=1
```

install
```
yum install -y nginx
```


모니터링을 위한 설정 추가
vi /etc/nginx/conf.d/default.conf
```
    location = /server_status {
        stub_status;
    }

```
